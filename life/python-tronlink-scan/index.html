python-tronlink-scan#

波场链的扫块功能,如果对于需要获取波场链上的交易情况,又没有条件部署一台自己的节点,那么采用本文的方式是一个不错的选择,通过调用接口也能及时获取到交易信息

安装需要的用到的插件

pip install -r requirements.txt

运行即可,有需要的可调整为数据库的结构来存

python main.py
